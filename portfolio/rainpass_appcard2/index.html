요약

  • 일반적인 앱카드와 달리 일회용 카드번호를 통신없이 자체 생성하여 사용하는 앱카드 시스템

상세기능

  • 오프라인에서 일회용 카드번호 생성
  • 일회용 카드번호가 해당 시스템의 어떤 사용자와도 겹치지 않는 유일성을 보장
  • 일회용 카드번호 유추 불가

개발환경

  • Android 4.4 이상, iOS 8 이상
  • Node.js 6, Vue.js 2.x
  • Android Studio 3.1, Xcode 9.2, Visual Studio Code 1.21
